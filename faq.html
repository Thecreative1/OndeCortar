Tens dúvidas? Encontra aqui as respostas mais comuns sobre o OndeCortar.pt.

O que é o OndeCortar.pt?

É um diretório de barbearias em Portugal com páginas de cidade, perfis individuais, loja e revista na mesma plataforma.

Como encontro uma barbearia?

Na homepage podes pesquisar por cidade, nome da barbearia ou morada. Também podes entrar pelas páginas de cidade e comparar perfis individuais.

Como posso adicionar uma barbearia ao OndeCortar.pt?

Podes usar o formulário de registo para enviar nome, responsável, email, telefone, cidade, morada, link de marcação e horários.

Quanto custa adicionar ou atualizar uma barbearia?

Nesta fase, a entrada base e a atualização de dados continuam sem custo para reforçar a cobertura e a qualidade do diretório.

Que dados podem aparecer num perfil?

Nome, morada, telefone, horários, website, Instagram, email e outras informações que a barbearia partilhar.

Como corrijo ou removo um perfil?

Podes usar o formulário de entrada/atualização ou enviar email para contacto@ondecortar.pt com a correção pretendida. Se fores responsável pelo espaço, também podes pedir revisão ou remoção dos dados públicos.

Como funciona a loja do OndeCortar.pt?

A loja mostra categorias, produtos recomendados e artigos ligados ao mesmo tema para ajudar a comparar antes de comprar.

Como funcionam os links afiliados?

Alguns links comerciais podem gerar comissão para o OndeCortar.pt quando existe compra elegível, sem custo adicional para o utilizador.

Os dados de uma barbearia são públicos?

A ficha pública mostra apenas os dados relevantes para descoberta e contacto, como nome, morada, telefone, horários e links úteis quando disponíveis.
