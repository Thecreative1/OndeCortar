Como funciona

O projeto cresce a partir da utilidade. Quanto mais fácil for descobrir uma boa barbearia, mais valor existe para utilizadores e para as próprias barbearias.

Descobre mais rápido

O objetivo é reduzir fricção. Menos cliques, menos confusão, mais probabilidade de encontrares um espaço útil perto de ti.

  • Pesquisa direta por nome ou localidade.
  • Links para Instagram, site ou mapa.
  • Horários e notas quando existirem.
