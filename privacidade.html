Privacidade

Esta página explica, de forma simples, como o OndeCortar.pt trata os dados enviados através do formulário de adesão e como funciona a transparência comercial do projeto.

Que dados podemos recolher

Quando uma barbearia preenche o formulário, podemos recolher dados como nome do negócio, nome do responsável, email, telefone, cidade, morada, links de presença online e informação adicional enviada na mensagem.

Para que usamos esses dados

Usamos os dados apenas para analisar o pedido, responder ao contacto, preparar um perfil no diretório ou apresentar uma proposta comercial relacionada com a entrada ou destaque da barbearia no OndeCortar.pt.

Ferramentas utilizadas

O formulário de contacto pode ser processado através do Formspree. Ao submeteres o pedido, reconheces que os dados podem passar por esse serviço para permitir a entrega da mensagem.

Links de afiliado

Algumas páginas da loja incluem links de afiliado para produtos externos. O OndeCortar.pt pode receber uma comissão se houver compra através desses links, sem custo adicional para o utilizador.

Contacto e direitos

Para pedir atualização, remoção ou esclarecimentos sobre os dados enviados, contacta contacto@ondecortar.pt.
